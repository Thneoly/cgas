### 3.1 BC-033: Batch 嵌套深度边界值

**问题描述**: 嵌套深度达到最大值 (5 层) 时，系统行为不稳定，偶发执行失败。

**根因分析**:
- 深度递归时栈空间不足
- 第 5 层嵌套结果聚合逻辑存在竞态条件

**修复方案**:
```rust
// 修复前: 递归深度无限制
fn execute_nested_batch(batch: Batch, depth: usize) -> Result {
    // 问题: 无深度检查
    execute_batch(batch)?;
}

// 修复后: 添加深度检查和栈空间优化
fn execute_nested_batch(batch: Batch, depth: usize) -> Result {
    const MAX_DEPTH: usize = 5;
    if depth > MAX_DEPTH {
        return Err(Error::NestedDepthExceeded { depth, max: MAX_DEPTH });
    }
    // 使用迭代替代递归，避免栈溢出
    execute_batch_iterative(batch, depth)?;
}
```

### 3.2 BC-034: Batch 嵌套原子性边界

**问题描述**: atomic=true 时，子 Batch 失败后父 Batch 未完全回滚。

**根因分析**:
- 回滚传播逻辑未正确处理嵌套场景
- 部分回滚状态未正确记录

**修复方案**:
```rust
// 修复前: 回滚未传播到父 Batch
fn execute_atomic_batch(batch: Batch) -> Result {
    for sub_batch in batch.nested {
        if let Err(e) = execute(sub_batch) {
            rollback(sub_batch)?; // 问题: 仅回滚当前
            return Err(e);
        }
    }
}

// 修复后: 回滚传播到所有层级
fn execute_atomic_batch(batch: Batch) -> Result {
    let mut execution_log = Vec::new();
    for sub_batch in batch.nested {
        match execute(sub_batch) {
            Ok(result) => execution_log.push(result),
            Err(e) => {
                rollback_all(&execution_log)?; // 修复: 回滚所有已执行
                return Err(e);
            }
        }
    }
}
```

| 深层嵌套失败 → 全部回滚 | 全部回滚 | ✅ 全部回滚 | ✅ PASS |

struct BatchContext {

struct BatchContext {
    variables: HashMap<String, Value>,

impl BatchContext {
    fn get_variable(&self, name: &str) -> Option<Value> {
        self.variables.get(name)
            .or_else(|| self.parent_context.as_ref()?.get_variable(name))

### 3.4 BC-036: Batch 嵌套资源清理

**问题描述**: 嵌套 Batch 执行后，临时资源未正确清理，导致内存泄漏。

**根因分析**:
- 资源清理逻辑未覆盖嵌套场景
- 异常路径下资源清理被跳过

**修复方案**:
```rust
// 修复前: 资源清理不完整
fn execute_batch(batch: Batch) -> Result {
    let resources = allocate_resources(&batch);
    let result = execute(&resources);
    // 问题: 异常时未清理
    cleanup(resources);
    result
}

// 修复后: 使用 RAII 模式确保清理
fn execute_batch(batch: Batch) -> Result {
    let resources = ResourceGuard::new(allocate_resources(&batch));
    // RAII: 无论成功失败都会清理
    execute(&resources)
}

struct ResourceGuard {
    resources: Resources,
}

impl Drop for ResourceGuard {
    fn drop(&mut self) {
        cleanup(&self.resources); // 自动清理
    }
}
```

- 错误类型转换丢失上下文信息
- 错误堆栈未正确传递

**修复方案**:
```rust
// 修复前: 错误信息丢失
fn execute_nested(batch: Batch) -> Result {
    match execute_sub_batch(batch) {
        Ok(r) => Ok(r),
        Err(e) => Err(Error::BatchFailed), // 问题: 丢失详情
    }
}

// 修复后: 错误链完整传播
fn execute_nested(batch: Batch) -> Result {
    execute_sub_batch(batch)
        .map_err(|e| Error::NestedBatchFailed {
            batch_id: batch.id,
            depth: batch.depth,
            source: Box::new(e), // 修复: 保留原始错误
        })
}
```
